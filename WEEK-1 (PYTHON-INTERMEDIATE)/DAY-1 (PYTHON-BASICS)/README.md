## 🛠️ Project: Contact Manager CLI

### Features Implemented

| # | Feature | Description |
|---|---------|-------------|
| 1 | **Add Contact** | Store name and phone number in memory |
| 2 | **View All Contacts** | Display all saved contacts |
| 3 | **Search Contact** | Find a contact by name |
| 4 | **Delete Contact** | Remove a contact from the system |
| 5 | **Exit** | Gracefully terminate the application |

### User Experience Flow

| Action | User Sees |
|--------|-----------|
| Menu | Clear numbered options displayed at start |
| Add | Confirmation message with contact name |
| View | Formatted list of all contacts |
| Search | Success message when contact found |
| Delete | Confirmation of successful deletion |
| Invalid Input | Friendly error message with prompt |
