## 🔍 Challenges & Solutions

| Challenge                               | Solution                                                     |
| --------------------------------------- | ------------------------------------------------------------ |
| Words with trailing spaces              | Used `.strip()` on all words before analysis                 |
| Counting only letters                   | Used `.isalpha()` to ignore numbers or symbols               |
| Correct longest/shortest word detection | Tracked length variables properly (`longest` and `shortest`) |
| Duplicate word detection                | Used frequency dictionary instead of repeated iteration      |
